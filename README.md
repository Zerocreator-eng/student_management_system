 Student Management System (Python CLI)

A simple command-line application built using Python to manage student records.  
This project demonstrates the use of object-oriented programming (OOP), data structures, and basic CRUD operations.
 🚀 Features

- ➕ Add a new student  
- 📋 View all students  
- 🔍 Search student by name  
- ✏️ Update student details  
- ❌ Delete a student  
- 💾 Data persistence using JSON (optional enhancement)
